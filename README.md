# false-certification.openbankruptcyproject.org

**False Certification Discharge**

Educational reference on the False Certification Discharge program for federal student loans, available where a school falsely certified a borrower's eligibility for federal student aid.

Live site: <https://false-certification.openbankruptcyproject.org/>

Maintained by the [Open Bankruptcy Project](https://openbankruptcyproject.org/), a 501(c)(3) public charity (EIN 41-5159631).
